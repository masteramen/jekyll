 2. 添加build path, 项目目录右键　Build Path -> Config build path -> java build Path ->　Libraries -> Add JARs 

 3. 把libs 文件夹下的jar包，全部添加上，再添加 selenium-java-2.44.0-src.jar和selenium-java-2.44.0.jar
